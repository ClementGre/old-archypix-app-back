# Roadmap for a MVP

## Pictures visualization

- Zoom in grid view
- View picture in full screen
- Ordering
- Upload button
- Reload button (clear cache & reload)

## Picture editing

- Batch editing in sidebar
- Saving exif when editing in sidebar
- Groups combo box allowing to manage manual groups or manage through SMC.
- Ratings
- Delete pictures

## Arrangements

- Allow manual group to have a filtering strategy, allowing them to be used inside of hierarchies.
- Support all filter types
- Support group by exif values/intervals
- Other group and deleted groups management (mapping references)

## Sharing

- Share groups with other users

## Hierarchy

- Create and configure hierarchy

## Syncing

- Serve a hierarchy file server with strong match conversion
